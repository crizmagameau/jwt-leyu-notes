# jwt-leyu-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖笔记、工具页面、静态演示等。这些页面均为独立文件，不依赖特定后端服务，适合用于本地浏览或静态托管。

## 目录说明

```
/
├── index.html          # 仓库索引页（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 页面引用的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── img/
└── README.md           # 本文件
```

- `pages/` 目录下为每个独立页面的 HTML 文件，命名尽量反映内容。
- `assets/` 目录存放页面所需的公用或专用资源文件。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，不针对任何具体网站或域名。页面内容可能包括：

- 个人学习笔记
- 工具型界面
- 静态演示或原型

每个页面可直接在浏览器中打开运行，部分页面可能需要网络加载外部资源（如字体、库文件）。

## 维护说明

- 欢迎通过 Issues 提交问题或建议。
- 若需新增页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 索引（如有）。
- 请勿将本仓库用于任何形式的商业用途或恶意目的。
- 仓库内容会不定期更新，不保证向后兼容。
